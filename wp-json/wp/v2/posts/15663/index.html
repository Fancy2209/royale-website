In this example, you can click the Jewel button to set up slider values. When you do this the ValueChangeEvent.VALUE_CHANGE<\/strong> will fire, calling the onValueChange<\/strong> event handler and setting width<\/em>, height<\/em> and text<\/em> in the Button<\/strong>.<\/p>\n

On the other hand, you can drag each slider<\/strong> to change width<\/em> and height<\/em> in the Button, and function to update value<\/em> in “onValueChange<\/em>” will be called continuously as you drag the slider controls.<\/p>\n

You can click in the track at any place to change the value immediately to the value at that point in the track. And if you need them, you have available “input<\/em>” and “change<\/em>” events in Slider. The first fires each time you move the slider thumb from one position to another, and the second fires when Slider ends its change from one position to another.<\/p>\n
