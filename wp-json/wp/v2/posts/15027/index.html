    \n
  • Application:<\/strong> Is the main tag in your application. All content in Apache Royale is added inside this tag.<\/li>\n
  • SimpleCSSValuesImpl:<\/strong> This class implements a minimal set of CSS rules that is sufficient for most applications.<\/li>\n
  • View:<\/strong> This is the class for most views in a Royale application.<\/li>\n
