With Apache Royale and Electron you can publish your projects to desktop applications. In this guide you’ll learn how to do that.<\/p>\n

The short version is to create an Electron project, create an Apache Royale project inside of that and then point the Electron app to the generated Apache Royale html page. You then communicate through the ipcRenderer. We’ll be using Visual Studio Code for this guide. The complete example project is attached at the end.<\/p>\n

    Create an Electron project<\/h3>\n

    A basic Electron application needs just these files:<\/p>\n

      \n
    • package.json<\/strong> – This points to the app’s main file and lists its details and dependencies.<\/li>\n
    • main.js<\/strong>\u00a0– Starts the app and creates a browser window to render HTML. This is the app’s\u00a0main process<\/strong>.<\/em><\/li>\n
    • index.html<\/strong>\u00a0– A web page to render. This is the app’s\u00a0renderer process<\/strong>.<\/li>\n<\/ul>\n

      You also need a reference to an Electron development instance. This can be in the form of a \/node_modules<\/strong> directory in your project or a reference to another external location. We will show two ways to reference it.<\/p>\n
