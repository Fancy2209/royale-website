Repositories<\/span><\/h2>[vc_column_text text_style=”leader-text”]Apache Royale is composed of the following repositories.[\/vc_column_text][\/vc_column][\/vc_row][vc_row bg_type=”color” padding_top_multiplier=”” bg_color=”#f9f9f9″][vc_column]

Compiler<\/span><\/h3>[vc_column_text text_style=”leader-text”]The Compiler is designed to transpile MXML and ActionScript to JavaScript, SWF and potentially other output definitions. We are working on other possible outputs like WebAssembly but many others could be as well added.<\/p>\n

You can check out the compiler source via git using the following commands:<\/p>\n

git clone \/\/github.com\/apache\/royale-compiler.git royale-compiler\r\ncd royale-compiler\r\ngit checkout develop<\/pre>\n
[\/vc_column_text][\/vc_column][\/vc_row][vc_row bg_type=”color” padding_top_multiplier=”” bg_color=”#f9f9f9″][vc_column]
Typedefs<\/span><\/h3>[vc_column_text text_style=”leader-text”]Typedefs are SWC files designed to allow type-checking for application development. They contain the type definitions in ActionScript, but none of the actual implementation of the Javascript code.<\/p>\n
These are used for type-checking at compile-time, but are not cross-compiled and included as Javascript code for deployment. To use these typedef swcs, the native Javascript code libraries are linked at run-time in the HTML.<\/p>\n
You can check out the typedefs source via git using the following commands:<\/p>\n
git clone \/\/github.com\/apache\/royale-typedefs.git royale-typedefs\r\ncd royale-typedefs\r\ngit checkout develop<\/pre>\n
[\/vc_column_text][\/vc_column][\/vc_row][vc_row bg_type=”color” padding_top_multiplier=”” padding_bottom_multiplier=”3x” bg_color=”#f9f9f9″][vc_column]
Framework (asjs)<\/span><\/h3>[vc_column_text text_style=”leader-text”]ASJS Framework is a collection of SWC files which can be used to make application development easier. This includes core language features in ActionScript which do not exist in Javascript, abstractions of HTML elements, component sets and more.<\/p>\n
Check out the asjs source via git using the following commands:<\/p>\n
git clone \/\/github.com\/apache\/royale-asjs.git royale-asjs\r\ncd royale-asjs\r\ngit checkout develop<\/pre>\n
